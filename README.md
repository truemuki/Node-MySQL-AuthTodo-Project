**Как запустить?**

Для запуска приложения требуется выполнить установку всех модулей с помощью npm:

```bash
npm i
```

БД MySql создается динамически при запуске

Создайте .env на основе .env.example и заполните данные.

Запустите проект:

```bash
npm run start
```
